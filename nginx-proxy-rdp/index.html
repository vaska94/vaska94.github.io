How to proxy RDP via Nginx

Publish date: 2020-09-02

Simply Use Snippet Bellow and thank me later :)

stream {
    server {
        listen     1337;
        proxy_pass 10.0.0.1:3389;
    }
    server {
        listen     1336;
        proxy_pass 10.0.0.2:3389;
    }
}
